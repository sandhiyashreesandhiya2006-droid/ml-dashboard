# 📊 Sales Analytics Dashboard with Future Sales Prediction

## 🚀 Project Overview

This project is an interactive Sales Analytics Dashboard developed using Python, Streamlit, Plotly, and Scikit-Learn.

The dashboard helps analyze sales performance, customer behavior, regional trends, payment methods, and predicts future sales using Machine Learning techniques.

---

## ✨ Features

✅ Total Sales Analysis

✅ Total Profit Analysis

✅ Monthly Sales Trend Visualization

✅ Monthly Profit Analysis

✅ Category-wise Sales Distribution

✅ Region-wise Sales Analysis

✅ Top 10 Customers Analysis

✅ Payment Method Contribution Analysis

✅ Future Sales Prediction using Machine Learning

✅ Download Dataset Option
